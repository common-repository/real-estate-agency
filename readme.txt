== Description ==
Transforme votre WP en site Internet pour agence immobiliere. Responsive pour mobile.
This plugin relying on a 3rd party as a service to manage back office. The servers are in region: europe-west and us-central for data and images.
This is a link to service : https://real-estate-france-db.appspot.com/managedb
Ce plugin permet de creer et de gerer un site Internet pour une agence immobiliere.
Fonctions de l\'application:
- Ajout, modification suppression de bien avec photos, descriptif, surface, lieu type de bien 
prix etc...
- Gestion des transferts des biens avec leurs photos au format XML vers d\'autre sites.
- Visualisation des prospects provenant du site web dedie.
- Gestion de l\'adresse des biens en mode cache ou ouvert suivant la situation.
- Gestion des proprietaires des biens en mode cache ou ouvert suivant la situation.

Vous pouvez visualiser les biens immobiliers de differents comptes en regroupant 
ces comptes en agence, en groupe d\'agences ou en reseaux d\'agences immobilieres a l\'image d\'une franchise immobiliere en connectant chaque compte a un compte. (chaque commercial(e) a un compte qu\'il peut connecter a un autre compte pour former ces differentes agences, groupes ou reseaux).

== Installation ==
- Telechargez les fichiers du plugin dans le repertoire `/ wp-content / plugins / plugin-name`, ou installez le plugin via l\'ecran des extensions WordPress directement.
- Activer le plugin via l\'ecran \'Extensions\' dans WordPrese
- Utilisez dans le menu administration le menu \"Tous les biens\" 
- Renseignez la page d\'enregistrement du logiciel a la premiere utilisation.
- Si vous avez deja un compte sur l\'application Facebook demander juste le mot de passe dans \"Mot de passe oublie\" sinon remplissez le formulaire.
- Parametrez la page Parametre smtp pour recevoir votre mot de passe en cas d oubli.
